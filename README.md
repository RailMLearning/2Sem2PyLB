# Отчет по лабораторной работе

## Описание задачи

В этой лабораторной работе нужно было реализовать набор функций для работы с
NumPy, pandas, matplotlib и seaborn. По заданию требовалось создавать массивы,
выполнять векторные и матричные операции, загружать данные из CSV, считать
основные статистики и строить несколько графиков. Дополнительно нужно было
подготовить тесты и убедиться, что все функции проходят проверку.

## Как я решил задачу

Сначала я реализовал все функции в `main.py` через стандартные возможности
NumPy и pandas. Для векторных и матричных операций использовал готовые
операции массивов, чтобы код был коротким и без лишних циклов. Для статистики
применил `mean`, `median`, `std`, `min`, `max` и `percentile`.

Для загрузки данных из CSV использовал `pandas.read_csv`, а затем перевёл
таблицу в массив NumPy. В части визуализации сделал сохранение графиков в
папку `plots`, чтобы результаты работы не терялись и их можно было посмотреть
отдельно. Также добавил безопасный backend `Agg`, потому что в моём окружении
не был доступен Tk, и из-за этого matplotlib падал при построении графиков.

После этого я проверил тесты и поправил их запуск, чтобы они работали через
локальное виртуальное окружение `numpy_env`.

## Нюансы при решении

Самый заметный нюанс был связан с matplotlib: по умолчанию он пытался открыть
GUI-бэкенд Tk, а в системе не хватало файлов Tcl/Tk. Из-за этого падал тест на
построение тепловой карты. Проблема решилась переходом на `Agg`, который
подходит для сохранения изображений без открытия окна.

## Вывод

В результате я получил рабочий набор функций для численных вычислений и
визуализации, а также тесты, которые успешно проходят. Лабораторная работа
выполнена полностью, а результаты можно воспроизвести через `pytest`.
